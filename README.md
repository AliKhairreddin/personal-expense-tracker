# Personal Expense Tracker

## Project Description
Personal Expense Tracker is a full-stack web application that helps students and young adults track expenses, manage budgets, and understand their spending habits.

## Main Features
- User accounts
- Add, view, edit, and delete transactions
- Categorize expenses
- Set budgets
- View spending summaries
- Track shared expenses
